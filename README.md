# Simuro 5v5 Robot Soccer Strategy
中国机器人及人工智能大赛 Simuro 仿真足球 5v5 队伍完整策略源码
队伍名：kimu队伍

## 📌 项目简介
本仓库为 Simuro 仿真足球 5v5 完整 Python 策略代码，适配官方 V5RPC 通信接口，用于中国机器人及人工智能大赛 Simuro 机器人竞技赛。
实现**多智能体分层决策、PID 运动控制、机器人位姿预测、球速线性预测、全场景战术逻辑**，覆盖正常比赛、开球、门球、点球、争球五大判罚场景，包含进攻、防守、守门员、突破、射门、绕球等全套行为。

### 核心技术亮点
1. **增量式 PID 双版本控制器**
   位置运动 PID、角度转向 PID、距离速度 PID，支持复位、历史误差缓存，机器人原地转向/直行倒车平滑调速。
2. **机器人运动状态预测**
   基于历史轮速、线速度、角速度建模，补偿通信延迟（tick_delay=4），解决感知滞后问题。
3. **足球一阶线性预测**
   `futureBall = 8*now -7*last` 线性外推预测球未来位置，提前预判攻防。
4. **五机器人角色分工体系**
   - 0号：主攻突破、射门
   - 1号：专职守门员，分区拦截、禁区防守
   - 2/3号：边路防守、中场拦截
   - 4号：中场自由人、补位协防
5. **完整判罚状态机**
   自动处理：开球、门球、点球、争球、点球大战，内置多套可切换战术决策树。
6. **自适应轮速调速**
   根据距离动态限制最大速度，角度差大于90°自动倒车行进，避免原地卡顿。
7. **定位球多套备选战术**
   点球5套进攻方案、门球3套进攻策略、攻防自动切换摆位逻辑。
